## Troubleshooting

### Problem: Can't see recent log
**Solution:** Refresh the page (F5) or check "View All Logs"

### Problem: Calendar not showing logs
**Solution:** Ensure logs are submitted with correct dates

### Problem: Skills not appearing
**Solution:** Add skills as comma-separated values in logs
Example: "PHP, MySQL, Web Development"

### Problem: Statistics not updating
**Solution:** Clear browser cache and refresh page

### Problem: Can't edit old log
**Solution:** Logs can only be edited within 24 hours of creation
